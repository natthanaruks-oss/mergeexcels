# MergeExcels v3.3.5 — Detached Thai Marks & Road Text Repair

## Fixed
- ยืนยันจากไฟล์ผลลัพธ์จริงว่าฟอนต์ PDF ส่ง Thai combining marks ออกมาเป็นเซลล์เดี่ยวหลายชนิด ไม่ได้มีเฉพาะนิคหิต `ํ`
- ล้างเซลล์ที่มีเฉพาะสระ/วรรณยุกต์ลอย เช่น `ุ`, `็้`, `ิุ์` เพื่อไม่ให้เกิดคอลัมน์วงกลมแปลก ๆ
- เพิ่ม Road Pack correction สำหรับคำที่สระ `ำ` หายซ้ำ เช่น `บารุง`, `กาหนด`, `อานวย`, `ตารวจ`
- ซ่อมชื่อจังหวัดหลัง `จ.` ด้วย Gazetteer แบบ normalized match เช่น `จ.กาแพงเพชร` → `จ.กำแพงเพชร`
- เพิ่ม regression tests จากรูปแบบที่พบในไฟล์ `pdf_to_excel (21).xlsx`

## Scope
- การเดาคำทำเฉพาะเมื่อเปิดโหมดเอกสารงานถนน
- เมนู 01–08 และ Optimize Excel ยังคงเดิม
